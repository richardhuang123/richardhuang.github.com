Best Practices for Modals / Overlays / Dialog Windows

After reading the article, I first learned what this mode is. First create a mode that disables the main window and make the main window and child windows visible. The user must first interact with the modal window before operating on the main window. Modal windows are very common on PC web pages. Because the screen on the PC side is relatively large, the modal windows are easy to identify. Unless you need to reconfirm the operation carefully, pop-up windows with background masks rarely appear. However, the space of the mobile screen is limited, which is why many modal windows on mobile devices occupy the entire screen. Therefore, the PC web modal window will be particularly advantageous.
